## Your setup checklist (required once)

1. `npx -y firebase-tools@latest login`
2. `dart pub global activate flutterfire_cli && flutterfire configure`
3. Firebase Console → Authentication → enable Email + Google
4. Firebase Console → Firestore → create database
5. `firebase deploy --only firestore:rules`
6. Sign in once → set `users/<uid>.role` = `super_admin` for your account
7. Add a location near your test GPS coordinates
8. `flutter run` on a **physical Android device**
